Projeto usado para otimizar meu tempo ao fazer merges necessários.



## Autores

## Instalação

Só tem duas dependencias esse projeto, que são elas

```bash
Ubuntu
    sudo apt install python3-tk
Windows 
    pip install tkinter    

pip install gitpython
```
    
## Stack utilizada

**Front-end:** tkinter

**Back-end:** python3
